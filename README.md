Responsive Video
================

### A simple way to make a iframe video responsive.

## How is made

* [HTML5 Reset](http://html5reset.org/) and [HTML5 Boilerplate](http://html5boilerplate.com) as base template.
* [960gs](http://960.gs/) for fluid layout.
